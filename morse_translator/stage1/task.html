[TITLE]Practice 87778[/TITLE]

Description

In this initial phase, the goal is to establish a visually intuitive and functional interface for users to interact with the Morse code translation application. The design process involves crafting a layout that neatly organizes input and output fields and buttons to work with the application.

Objectives

Create a layout that includes:

  1. An EditText with an id of et_text for users to enter text that will be translated to morse. Hint should be set to "Text".

  2. An EditText with an id of et_morse to display the translated text or entering a Morse code. Hint should be set to "Morse".

  3. A button with an id of bt_clear to clear both EditTexts.

  4. A button with an id of bt_guide that will be used to open an activity with a dictionary and an instruction for the app.

  5. A TextView with an id of tv_text with a "Text" set as a text and placed near the et_text EditText

  6. A TextView with an id of tv_morse with a "Morse" set as a text and placed near the et_morse EditText

No actions are required for this stage, just define the Layout with correct ID's and Views.
